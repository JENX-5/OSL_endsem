Q1 LSTF Disk Scheduling + MFU Page Replacement
▼
LSTF (Largest Seek Time First) is a contrarian disk scheduling policy that services the cylinder requiring the largest seek distance first — useful to study as a worst-case contrast. The resulting visit order becomes the page reference string, which is then processed with MFU (Most Frequently Used) page replacement — evicts the most frequently accessed page.
LSTF is the opposite of SSTF. MFU is the opposite of LFU. Both are rarely used in practice but common in exams to test understanding of extremes.

Sample Input

Frames: 3
Head: 50
Requests (page cylinder):
  1 82
  2 170
  3 43
  4 140
  5 24
  -1 (stop)

Sample Output

LSTF Seek Sequence:
  170 → 43 → 140 → 24 → 82
Total Seek Time: 481

MFU Page Replacement:
  Step 1: Ref 2 → FAULT [ 2 - - ]
  Step 2: Ref 4 → FAULT [ 2 4 - ]
  ...
Faults: 4 | Hits: 1
Hit Ratio: 20.00%
q1_lstf_mfu.c 
/*
 * Q1: LSTF (Largest Seek Time First) Disk Scheduling
 *     + MFU (Most Frequently Used) Page Replacement
 *
 * Flow:
 *   1. Read (page_number, cylinder) pairs
 *   2. Sort by |cylinder - head| descending → LSTF order
 *   3. Use the LSTF visit order as the page reference string
 *   4. Run MFU page replacement on that string
 */

#include <stdio.h>
#include <stdlib.h>
#include <math.h>
#include <string.h>

#define MAX_REQ 100

/* ── data structures ─────────────────────────────── */
typedef struct {
    int page;       /* logical page number          */
    int cylinder;   /* physical cylinder position   */
    int seek_dist;  /* |cylinder - current_head|    */
} Request;

typedef struct {
    int page;
    int freq;       /* access frequency             */
    int last_used;  /* timestamp of last use        */
} Frame;

/* ── helpers ─────────────────────────────────────── */

/* Compare for descending seek distance (LSTF) */
int cmp_lstf(const void *a, const void *b) {
    return ((Request *)b)->seek_dist - ((Request *)a)->seek_dist;
}

/* Print current frame state */
void print_frames(Frame *frames, int nframes) {
    printf("  Frames: [ ");
    for (int i = 0; i < nframes; i++) {
        if (frames[i].page == -1) printf("- ");
        else printf("%d ", frames[i].page);
    }
    printf("]");
}

/* ── LSTF Disk Scheduling ────────────────────────── */
int lstf(Request *reqs, int n, int head, int *seek_seq) {
    /* compute seek distances from initial head */
    for (int i = 0; i < n; i++)
        reqs[i].seek_dist = abs(reqs[i].cylinder - head);

    /* sort descending by seek distance */
    qsort(reqs, n, sizeof(Request), cmp_lstf);

    int total = 0, cur = head;
    printf("\n=== LSTF Disk Scheduling ===\n");
    printf("%-6s %-10s %-10s %-12s\n",
           "Step", "Page", "Cylinder", "Seek Dist");
    printf("--------------------------------------\n");

    for (int i = 0; i < n; i++) {
        int dist = abs(reqs[i].cylinder - cur);
        total += dist;
        seek_seq[i] = reqs[i].page;   /* page reference string */
        printf("%-6d %-10d %-10d %-12d\n",
               i + 1, reqs[i].page, reqs[i].cylinder, dist);
        cur = reqs[i].cylinder;
    }
    printf("--------------------------------------\n");
    printf("Total Seek Time: %d\n\n", total);
    return total;
}

/* ── MFU Page Replacement ────────────────────────── */
void mfu(int *ref_str, int n, int nframes) {
    Frame frames[MAX_REQ];
    for (int i = 0; i < nframes; i++) { frames[i].page = -1; frames[i].freq = 0; frames[i].last_used = -1; }

    int faults = 0, hits = 0, loaded = 0, time = 0;

    printf("=== MFU Page Replacement ===\n");
    printf("Frames=%d | Reference String Length=%d\n\n", nframes, n);
    printf("%-6s %-8s %-8s %-8s", "Step", "Ref", "Hit/F", "Evicted");
    print_frames(frames, nframes); printf("\n");
    printf("----------------------------------------------------\n");

    for (int i = 0; i < n; i++, time++) {
        int page = ref_str[i];
        int found = 0, free_slot = -1, evict = -1;

        /* check hit */
        for (int j = 0; j < nframes; j++) {
            if (frames[j].page == page) {
                frames[j].freq++;
                frames[j].last_used = time;
                found = 1; hits++;
                break;
            }
        }

        if (!found) {
            faults++;
            /* find free slot */
            for (int j = 0; j < nframes; j++)
                if (frames[j].page == -1) { free_slot = j; break; }

            if (free_slot != -1) {
                /* load into empty frame */
                frames[free_slot].page = page;
                frames[free_slot].freq = 1;
                frames[free_slot].last_used = time;
                evict = -1;
            } else {
                /* MFU: evict frame with HIGHEST frequency
                   tie-break: least recently used */
                int max_freq = -1, lru_time = time + 1;
                for (int j = 0; j < nframes; j++) {
                    if (frames[j].freq > max_freq ||
                       (frames[j].freq == max_freq && frames[j].last_used < lru_time)) {
                        max_freq = frames[j].freq;
                        lru_time = frames[j].last_used;
                        evict = j;
                    }
                }
                int evicted_page = frames[evict].page;
                frames[evict].page = page;
                frames[evict].freq = 1;
                frames[evict].last_used = time;
                printf("%-6d %-8d %-8s %-8d",
                       i + 1, page, "FAULT", evicted_page);
                print_frames(frames, nframes); printf("\n");
                continue;
            }
        }

        printf("%-6d %-8d %-8s %-8s",
               i + 1, page, found ? "HIT" : "FAULT", "-");
        print_frames(frames, nframes); printf("\n");
    }

    printf("----------------------------------------------------\n");
    printf("Total Page Faults : %d\n", faults);
    printf("Total Page Hits   : %d\n", hits);
    printf("Hit Ratio         : %.2f%%\n", (float)hits / n * 100);
}

/* ── main ────────────────────────────────────────── */
int main(void) {
    int nframes, head;
    Request reqs[MAX_REQ];
    int n = 0;

    printf("=== Q1: LSTF + MFU ===\n\n");
    printf("Enter number of frames: "); scanf("%d", &nframes);
    printf("Enter initial head position: "); scanf("%d", &head);
    printf("Enter (page cylinder) pairs, -1 to stop:\n");

    while (1) {
        int pg, cyl;
        scanf("%d", &pg);
        if (pg == -1) break;
        scanf("%d", &cyl);
        reqs[n].page = pg;
        reqs[n].cylinder = cyl;
        n++;
    }

    int seek_seq[MAX_REQ];
    lstf(reqs, n, head, seek_seq);
    mfu(seek_seq, n, nframes);
    return 0;
}
Q2 LCFS Disk Scheduling + LFU Page Replacement
▼
LCFS (Last Come First Serve) processes disk requests in reverse arrival order — like a stack (LIFO). LFU (Least Frequently Used) page replacement evicts the page with the lowest access frequency; ties broken by LRU time.
LCFS is also called "LIFO Disk Scheduling". The most recently arrived request is serviced first — it can starve older requests.

Sample Input

Frames: 3
Head: 100
Requests (page cylinder):
  1 50
  2 30
  3 90
  4 150
  5 10
  -1

Sample Output

LCFS (reverse arrival):
  10 → 150 → 90 → 30 → 50
Total Seek Time: 420

LFU Simulation:
  Each fault → load page
  Evict lowest freq on fault
  Hit updates frequency

/*
 * Q2: LCFS (Last Come First Serve) Disk Scheduling
 *     + LFU (Least Frequently Used) Page Replacement
 *
 * LCFS = process requests in REVERSE arrival order
 *        (stack / LIFO discipline)
 */

#include <stdio.h>
#include <stdlib.h>
#include <math.h>

#define MAX_REQ 100

/* ── data structures ─────────────────────────────── */
typedef struct {
    int page;
    int cylinder;
} Request;

typedef struct {
    int page;
    int freq;
    int last_used;
} Frame;

/* ── LCFS Disk Scheduling ────────────────────────── */
int lcfs(Request *reqs, int n, int head, int *seek_seq) {
    int total = 0, cur = head;

    printf("\n=== LCFS Disk Scheduling ===\n");
    printf("Arrival order reversed (LIFO / stack)\n\n");
    printf("%-6s %-10s %-10s %-12s\n",
           "Step", "Page", "Cylinder", "Seek Dist");
    printf("--------------------------------------\n");

    /* process in reverse arrival order */
    for (int i = n - 1; i >= 0; i--) {
        int dist = abs(reqs[i].cylinder - cur);
        total += dist;
        seek_seq[n - 1 - i] = reqs[i].page;
        printf("%-6d %-10d %-10d %-12d\n",
               n - i, reqs[i].page, reqs[i].cylinder, dist);
        cur = reqs[i].cylinder;
    }
    printf("--------------------------------------\n");
    printf("Total Seek Time: %d\n\n", total);
    return total;
}

/* ── LFU Page Replacement ────────────────────────── */
void print_frames_lfu(Frame *frames, int nf) {
    printf("  [ ");
    for (int i = 0; i < nf; i++) {
        if (frames[i].page == -1) printf("- ");
        else printf("%d(f%d) ", frames[i].page, frames[i].freq);
    }
    printf("]");
}

void lfu(int *ref_str, int n, int nframes) {
    Frame frames[MAX_REQ];
    for (int i = 0; i < nframes; i++) {
        frames[i].page = -1; frames[i].freq = 0; frames[i].last_used = -1;
    }

    int faults = 0, hits = 0, time = 0;

    printf("=== LFU Page Replacement ===\n");
    printf("Frames=%d  (format: page(freq))\n\n", nframes);
    printf("%-6s %-6s %-8s %-8s  Frames\n", "Step", "Ref", "Hit/F", "Evicted");
    printf("----------------------------------------------------\n");

    for (int i = 0; i < n; i++, time++) {
        int page = ref_str[i];
        int found = 0, free_slot = -1, evict = -1;

        for (int j = 0; j < nframes; j++) {
            if (frames[j].page == page) {
                frames[j].freq++;
                frames[j].last_used = time;
                found = 1; hits++;
                break;
            }
        }

        if (!found) {
            faults++;
            for (int j = 0; j < nframes; j++)
                if (frames[j].page == -1) { free_slot = j; break; }

            if (free_slot != -1) {
                frames[free_slot].page = page;
                frames[free_slot].freq = 1;
                frames[free_slot].last_used = time;
                printf("%-6d %-6d %-8s %-8s", i+1, page, "FAULT", "-");
            } else {
                /* LFU: evict page with LOWEST frequency; tie → LRU */
                int min_freq = 1<<30, lru_t = time+1;
                for (int j = 0; j < nframes; j++) {
                    if (frames[j].freq < min_freq ||
                       (frames[j].freq == min_freq && frames[j].last_used < lru_t)) {
                        min_freq = frames[j].freq;
                        lru_t    = frames[j].last_used;
                        evict    = j;
                    }
                }
                int ep = frames[evict].page;
                frames[evict].page = page;
                frames[evict].freq = 1;
                frames[evict].last_used = time;
                printf("%-6d %-6d %-8s %-8d", i+1, page, "FAULT", ep);
            }
        } else {
            printf("%-6d %-6d %-8s %-8s", i+1, page, "HIT", "-");
        }
        print_frames_lfu(frames, nframes); printf("\n");
    }

    printf("----------------------------------------------------\n");
    printf("Faults: %d | Hits: %d | Hit Ratio: %.2f%%\n",
           faults, hits, (float)hits/n*100);
}

/* ── main ────────────────────────────────────────── */
int main(void) {
    int nframes, head;
    Request reqs[MAX_REQ];
    int n = 0;

    printf("=== Q2: LCFS + LFU ===\n\n");
    printf("Enter number of frames: "); scanf("%d", &nframes);
    printf("Enter initial head position: "); scanf("%d", &head);
    printf("Enter (page cylinder) pairs, -1 to stop:\n");

    while (1) {

        if (pg == -1) break;
        int cyl; scanf("%d", &cyl);

    }

    int seek_seq[MAX_REQ];
    lcfs(reqs, n, head, seek_seq);
    lfu(seek_seq, n, nframes);
    return 0;
}

Q4 MRU Page Replacement using 8-bit Reference Registers
▼
Each frame has an 8-bit shift register. On every memory reference:
  1. Shift ALL frame registers right by 1 bit
  2. Set MSB = 1 for the referenced page's frame
The frame with the maximum register value is the MRU (Most Recently Used) page — evict it on a page fault.
Register after reference = 10000000 (binary) = 0x80
Register after 1 miss = 01000000
Register after 2 misses = 00100000
Higher value = more recently referenced = MRU victim

Sample Input

Frames: 3
References: 5
String: 1 2 3 2 1

Sample Output

Step 1 | Ref:1 | FAULT
  Frames: [1 - -]
  Regs: [10000000 00000000 00000000]

Step 2 | Ref:2 | FAULT
  Frames: [1 2 -]
  Regs: [01000000 10000000 00000000]

Step 4 | Ref:2 | HIT
  ...Evict page with max reg...
q4_mru_registers.c 
/*
 * Q4: MRU Page Replacement using 8-bit Reference Registers
 *
 * Each frame maintains an 8-bit shift register.
 * On every memory reference:
 *   1. Shift ALL frame registers RIGHT by 1.
 *   2. Set MSB (bit 7) = 1 for the referenced page.
 *
 * Replacement: evict the frame with the MAXIMUM register value
 * (i.e., the Most Recently Used page).
 * Tie-break: first frame found with max value.
 */

#include <stdio.h>
#include <string.h>

#define MAX_FRAMES  10
#define MAX_REF    100

/* ── helpers ─────────────────────────────────────── */

/* Print 8-bit value as binary string */
void to_binary8(unsigned char val, char *buf) {
    for (int i = 7; i >= 0; i--)
        buf[7 - i] = ((val >> i) & 1) ? '1' : '0';
    buf[8] = '\0';
}

/* Find page in frames; return index or -1 */
int find_page(int *frames, int nf, int page) {
    for (int i = 0; i < nf; i++)
        if (frames[i] == page) return i;
    return -1;
}

/* Find frame with MAXIMUM register (MRU victim) */
int find_mru_victim(unsigned char *regs, int nf) {
    int victim = 0;
    for (int i = 1; i < nf; i++)
        if (regs[i] > regs[victim]) victim = i;
    return victim;
}

/* Find free frame (-1 if none) */
int find_free(int *frames, int nf) {
    for (int i = 0; i < nf; i++)
        if (frames[i] == -1) return i;
    return -1;
}

/* Print current frame & register state */
void print_state(int *frames, unsigned char *regs, int nf) {
    char buf[9];
    printf("  Frames : [ ");
    for (int i = 0; i < nf; i++) {
        if (frames[i] == -1) printf("- "); else printf("%d ", frames[i]);
    }
    printf("]\n");
    printf("  Regs   : [ ");
    for (int i = 0; i < nf; i++) {
        to_binary8(regs[i], buf);
        printf("%s ", buf);
    }
    printf("]\n");
}

/* ── MRU simulation ──────────────────────────────── */
void mru_simulate(int *ref_str, int n, int nframes) {
    int frames[MAX_FRAMES];
    unsigned char regs[MAX_FRAMES];

    memset(frames, -1, sizeof(frames));
    memset(regs,    0, sizeof(regs));

    int faults = 0, hits = 0;

    printf("\n=== MRU Page Replacement (8-bit Registers) ===\n");
    printf("Frames = %d\n\n", nframes);

    for (int step = 0; step < n; step++) {
        int page = ref_str[step];
        printf("Step %2d | Ref: %d\n", step + 1, page);

        /* Step 1: shift ALL registers right by 1 */
        for (int i = 0; i < nframes; i++) regs[i] >>= 1;

        int idx = find_page(frames, nframes, page);

        if (idx != -1) {
            /* HIT: set MSB of referenced frame */
            regs[idx] |= 0x80;
            hits++;
            printf("  Result : HIT  (frame %d)\n", idx);
        } else {
            faults++;
            int free_slot = find_free(frames, nframes);

            if (free_slot != -1) {
                /* Load into empty frame */
                frames[free_slot] = page;
                regs[free_slot]   = 0x80;     /* MSB = 1 */
                printf("  Result : PAGE FAULT (loaded into frame %d)\n", free_slot);
            } else {
                /* MRU eviction */
                int victim = find_mru_victim(regs, nframes);
                printf("  Result : PAGE FAULT (evict page %d from frame %d)\n",
                       frames[victim], victim);
                frames[victim] = page;
                regs[victim]   = 0x80;
            }
        }

        print_state(frames, regs, nframes);
        printf("\n");
    }

    printf("=== Summary ===\n");
    printf("Total Steps  : %d\n", n);
    printf("Total Hits   : %d\n", hits);
    printf("Total Faults : %d\n", faults);
    printf("Hit Ratio    : %.2f%%\n", (float)hits / n * 100);
}

/* ── main ────────────────────────────────────────── */
int main(void) {
    int nframes, n;
    int ref_str[MAX_REF];

    printf("=== Q4: MRU with 8-bit Reference Registers ===\n\n");
    printf("Enter number of frames (max %d): ", MAX_FRAMES);
    scanf("%d", &nframes);

    printf("Enter number of references: ");
    scanf("%d", &n);

    printf("Enter reference string:\n");
    for (int i = 0; i < n; i++) {
        printf("  Ref[%d]: ", i); scanf("%d", &ref_str[i]);
    }

    mru_simulate(ref_str, n, nframes);
    return 0;
}
Q5 Memory Allocation (Best Fit) + Banker's Safety Check
▼
For each allocation request: find the Best Fit memory block (smallest block ≥ request), perform the allocation, then run the Banker's Safety Algorithm to verify the system remains in a safe state. Print the safe sequence if safe, or warn of unsafe state.
Best Fit: find min(block.size) where block.size >= request
Safety: Work = Available
  For each unfinished process P where Need[P] <= Work:
    Work += Allocation[P]; mark P finished
  Repeat until no progress. Safe iff all finished.

Sample Input

Processes: 3, Blocks: 4
Blocks: 100 500 200 300
Max: P0=200, P1=400, P2=150
Requests:
  P0 150
  P1 350
  -1

Sample Output

Request: P0 needs 150
  Before: B0(100) B1(500) B2(200) B3(300)
  Best Fit: B2(200) → P0
  After: B2 has 50 remaining
  Safety: SAFE ✓
  Sequence: P0 → P2 → P1
q5_bestfit_banker.c 
/*
 * Q5: Memory Allocation — Best Fit + Banker's Safety Check
 *
 * For each allocation request:
 *   1. Display memory blocks (sorted ascending for best-fit scan)
 *   2. Apply Best Fit: smallest block that fits
 *   3. Display memory after allocation
 *   4. Run simplified Banker's safety check
 *   5. Print safe sequence if safe, "UNSAFE" otherwise
 */

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define MAX_PROC   20
#define MAX_BLOCKS 20

/* ── memory block structure ──────────────────────── */
typedef struct {
    int id;         /* original block index */
    int size;       /* current free size     */
    int allocated_to; /* -1 = free            */
} Block;

/* ── Banker's safety check ───────────────────────── */
/*
 * Simplified: one resource type.
 * allocation[i] = currently allocated to process i
 * max_need[i]   = maximum need of process i
 * available     = total free memory
 */
int is_safe(int *allocation, int *max_need, int np, int available,
            int *safe_seq, int *seq_len) {
    int need[MAX_PROC], finish[MAX_PROC];
    for (int i = 0; i < np; i++) {
        need[i]   = max_need[i] - allocation[i];
        finish[i] = 0;
    }

    int work = available, count = 0;
    *seq_len = 0;

    while (count < np) {
        int found = 0;
        for (int i = 0; i < np; i++) {
            if (!finish[i] && need[i] <= work) {
                work += allocation[i];
                finish[i] = 1;
                safe_seq[(*seq_len)++] = i;
                found = 1;
                count++;
            }
        }
        if (!found) return 0;   /* unsafe */
    }
    return 1;   /* safe */
}

/* ── print memory blocks ─────────────────────────── */
void print_blocks(Block *blocks, int nb) {
    printf("  +-------+----------+------------------+\n");

/*
 * Q6: Optimal Page Replacement (Bélády's Algorithm)
 *
 * Replacement policy: evict the page that will NOT be used
 * for the LONGEST time in the future.
 * If a page is never used again, it is the best victim.
 */

#include <stdio.h>
#include <string.h>

#define MAX_FRAMES 10
#define MAX_REF   100

/* ── helpers ─────────────────────────────────────── */

int find_page(int *frames, int nf, int page) {
    for (int i = 0; i < nf; i++) if (frames[i] == page) return i;
    return -1;
}

int find_free(int *frames, int nf) {
    for (int i = 0; i < nf; i++) if (frames[i] == -1) return i;
    return -1;
}

/* Find optimal victim: page used FARTHEST in future (or never) */
int find_optimal_victim(int *frames, int nf, int *ref_str, int n, int cur) {
    int farthest = -1, victim = -1;

    for (int i = 0; i < nf; i++) {
        int next_use = n;   /* assume never used = n (beyond end) */
        for (int j = cur + 1; j < n; j++) {
            if (ref_str[j] == frames[i]) { next_use = j; break; }
        }
        if (next_use > farthest) { farthest = next_use; victim = i; }
    }
    return victim;
}

    printf("[ ");
    for (int i = 0; i < nf; i++) {

    }
    printf("]");
}

        } else {

            if (free_slot != -1) {
                frames[free_slot] = page;
                printf("%-6d %-6d %-8s %-10s  ", i+1, page, "FAULT", "-");
            } else {

                printf("%-6d %-6d %-8s %-10d  ", i+1, page, "FAULT", frames[victim]);
                frames[victim] = page;

        print_frames(frames, nframes);
        printf("\n");
    }

    printf("-----------------------------------------------\n");
    printf("Total Page Faults : %d\n", faults);
    printf("Total Page Hits   : %d\n", hits);
    printf("Hit Ratio         : %.2f%%\n", (float)hits / n * 100);

Q7 Effective Access Time (EAT) with TLB Hit Ratio
▼
EAT measures the average time per memory access accounting for TLB hits and misses. A TLB hit requires one memory access (after TLB lookup); a miss requires two (page table walk + data fetch).
EAT = α × (t_TLB + t_mem) + (1-α) × (t_TLB + 2×t_mem)

Simplified: EAT = t_TLB + (2 - α) × t_mem

Where: α = TLB hit ratio, t_TLB = TLB access time, t_mem = memory access time

Sample Input

TLB access time: 10 ns
Memory access time: 100 ns
Hit ratio: 0.90

Sample Output

EAT = 0.90 × (10 + 100)
    + 0.10 × (10 + 200)
    = 0.90 × 110 + 0.10 × 210
    = 99 + 21
    = 120 ns

/*
 * Q7: Effective Access Time (EAT) Calculation
 *
 * Formulas:
 *   EAT = α × (TLB_time + Mem_time)          ← TLB hit
 *       + (1 - α) × (TLB_time + 2 × Mem_time) ← TLB miss (2 memory accesses)
 *
 *   Where α = TLB hit ratio (0.0 – 1.0)
 *
 * With TLB associative lookup time = t_tlb ns
 * Memory access time               = t_mem ns
 *
 * Expanded:
 *   EAT = α × (t_tlb + t_mem) + (1-α) × (t_tlb + 2×t_mem)
 *       = t_tlb + t_mem + (1-α) × t_mem
 *       = t_tlb + (2 - α) × t_mem
 */

#include <stdio.h>

/* ── EAT calculation ─────────────────────────────── */
double compute_eat(double tlb_time, double mem_time, double hit_ratio) {
    double hit_path  = tlb_time + mem_time;           /* 1 memory access */
    double miss_path = tlb_time + 2.0 * mem_time;     /* 2 memory accesses */
    return hit_ratio * hit_path + (1.0 - hit_ratio) * miss_path;
}

int main(void) {

    printf("Enter Memory access time (ns): "); scanf("%lf", &mem_time);
    printf("Enter TLB hit ratio (0–1)   : "); scanf("%lf", &hit_ratio);

    double eat = compute_eat(tlb_time, mem_time, hit_ratio);

    hline();

    printf("  EAT = %.2f × %.2f + %.2f × %.2f\n",
           hit_ratio,       tlb_time + mem_time,
           1.0 - hit_ratio, tlb_time + 2.0 * mem_time);
    printf("  EAT = %.4f ns\n\n", eat);

    hline();

    printf("\nNOTE — Multi-level Paging:\n");
    printf("  For k-level page table:\n");
    printf("    EAT (no TLB)  = (k+1) × Mem_time\n");
    printf("    EAT (with TLB)= α × (TLB + Mem)\n");
    printf("                  + (1-α) × (TLB + (k+1)×Mem)\n");

    return 0;
}

    int left[MAX_REQ], right[MAX_REQ], nl = 0, nr = 0;

            }
            frames[vic] = page; last_used[vic] = i;
            printf("%-6d %-6d %-8s", i+1, page, "FAULT");
        }
        printf("  [ "); for (int j = 0; j < nf; j++) { if (frames[j]==-1) printf("- "); else printf("%d ", frames[j]); } printf("]\n");
    }
    printf("--------------------------------------\n");
    printf("Faults: %d | Hits: %d | Hit Ratio: %.2f%%\n", faults, hits, (float)hits/n*100);
}

/* ================================================================
 * Q13: Second Chance (Clock) Page Replacement
 * ================================================================
 * Each frame has a reference bit (R).
 * On fault: scan frames in FIFO order.
 *   If R=0 → evict this frame.
 *   If R=1 → clear R, advance pointer (second chance).
 *
 * Sample Input: Frames=3, Ref: 1 2 3 4 1 2 5 1 2 3 4 5
 * ================================================================ */
void second_chance(void) {
    int nf, n, ref[MAX_REQ];
    printf("\n=== Q13: Second Chance (Clock) Page Replacement ===\n");
    printf("Enter frames: "); scanf("%d", &nf);
    printf("Enter ref count: "); scanf("%d", &n);
    printf("Enter ref string:\n");
    for (int i = 0; i < n; i++) { printf("[%d]: ", i); scanf("%d", &ref[i]); }

    int frames[MAX_FRAMES], rbits[MAX_FRAMES];
    for (int i = 0; i < nf; i++) { frames[i] = -1; rbits[i] = 0; }

    int faults = 0, hits = 0, ptr = 0;
    printf("\n%-6s %-6s %-8s  Frames + R-bits\n", "Step", "Ref", "Hit/F");
    printf("----------------------------------------------\n");

    for (int i = 0; i < n; i++) {
        int page = ref[i], found = -1;
        for (int j = 0; j < nf; j++) if (frames[j] == page) { found = j; break; }

        if (found != -1) {
            hits++; rbits[found] = 1;
            printf("%-6d %-6d %-8s", i+1, page, "HIT");
        } else {
            faults++;
            /* clock algorithm */
            while (1) {
                if (rbits[ptr] == 0) break;
                rbits[ptr] = 0;
                ptr = (ptr + 1) % nf;
            }
            frames[ptr] = page; rbits[ptr] = 1;
            ptr = (ptr + 1) % nf;
            printf("%-6d %-6d %-8s", i+1, page, "FAULT");
        }
        printf("  [ ");
        for (int j = 0; j < nf; j++) {
            if (frames[j]==-1) printf("-,R=0 ");
            else printf("%d,R=%d ", frames[j], rbits[j]);
        }
        printf("]\n");
    }
    printf("----------------------------------------------\n");
    printf("Faults: %d | Hits: %d | Hit Ratio: %.2f%%\n", faults, hits, (float)hits/n*100);
}

/* ================================================================
 * Q14: Banker's Algorithm — Full Multi-Resource
 * ================================================================
 * Safety Algorithm + Resource Request Algorithm.
 *
 * Sample Input:
 *   Processes: 5, Resources: 3 (A, B, C)
 *   Allocation:
 *     P0: 0 1 0  P1: 2 0 0  P2: 3 0 2  P3: 2 1 1  P4: 0 0 2
 *   Max:
 *     P0: 7 5 3  P1: 3 2 2  P2: 9 0 2  P3: 2 2 2  P4: 4 3 3
 *   Available: 3 3 2
 * ================================================================ */
void bankers(void) {
    int np, nr;
    int alloc[MAX_PROC][MAX_RES], max_m[MAX_PROC][MAX_RES];
    int need[MAX_PROC][MAX_RES], avail[MAX_RES];

    printf("\n=== Q14: Banker's Algorithm (Multi-Resource) ===\n");
    printf("Enter number of processes: "); scanf("%d", &np);
    printf("Enter number of resources: "); scanf("%d", &nr);

    printf("Enter Allocation matrix (%d x %d):\n", np, nr);
    for (int i = 0; i < np; i++) {
        printf("  P%d: ", i);
        for (int j = 0; j < nr; j++) scanf("%d", &alloc[i][j]);
    }

    printf("Enter Max matrix (%d x %d):\n", np, nr);
    for (int i = 0; i < np; i++) {
        printf("  P%d: ", i);
        for (int j = 0; j < nr; j++) { scanf("%d", &max_m[i][j]); need[i][j] = max_m[i][j] - alloc[i][j]; }
    }

    printf("Enter Available resources: ");
    for (int j = 0; j < nr; j++) scanf("%d", &avail[j]);

    /* ── Print Need matrix ── */
    printf("\nNeed Matrix (Max - Allocation):\n");
    for (int i = 0; i < np; i++) {
        printf("  P%d: ", i);
        for (int j = 0; j < nr; j++) printf("%3d", need[i][j]);
        printf("\n");
    }

    /* ── Safety Algorithm ── */
    int work[MAX_RES], finish[MAX_PROC] = {0};
    for (int j = 0; j < nr; j++) work[j] = avail[j];

    int safe_seq[MAX_PROC], seq_len = 0;
    int count = 0;

    while (count < np) {
        int found = 0;
        for (int i = 0; i < np; i++) {
            if (finish[i]) continue;
            int ok = 1;
            for (int j = 0; j < nr; j++) if (need[i][j] > work[j]) { ok = 0; break; }
            if (ok) {
                for (int j = 0; j < nr; j++) work[j] += alloc[i][j];
                finish[i] = 1;
                safe_seq[seq_len++] = i;
                found = 1; count++;
            }
        }
        if (!found) break;
    }

    if (seq_len == np) {
        printf("\nSystem is in SAFE STATE ✓\nSafe Sequence: ");
        for (int i = 0; i < np; i++) printf("P%d%s", safe_seq[i], i<np-1?" → ":"");
        printf("\n");
    } else {
        printf("\nSystem is in UNSAFE STATE ✗ — Deadlock possible!\n");
    }

    /* ── Resource Request ── */
    printf("\nDo you want to simulate a resource request? (1=Yes/0=No): ");
    int choice; scanf("%d", &choice);
    if (!choice) return;

    int pid, req[MAX_RES];
    printf("Process ID requesting: "); scanf("%d", &pid);
    printf("Request vector: ");
    for (int j = 0; j < nr; j++) scanf("%d", &req[j]);

    /* Check: req <= need */
    for (int j = 0; j < nr; j++) {
        if (req[j] > need[pid][j]) { printf("Error: Request exceeds declared max need!\n"); return; }
    }
    /* Check: req <= available */
    for (int j = 0; j < nr; j++) {
        if (req[j] > avail[j]) { printf("P%d must wait — resources not available.\n", pid); return; }
    }
    /* Tentatively allocate */
    for (int j = 0; j < nr; j++) {
        avail[j]       -= req[j];
        alloc[pid][j]  += req[j];
        need[pid][j]   -= req[j];
    }
    printf("Tentative allocation done. Re-running safety check...\n");
    /* re-run safety (recursive call would be cleaner; inline for simplicity) */
    for (int j = 0; j < nr; j++) work[j] = avail[j];
    memset(finish, 0, sizeof(finish));
    seq_len = 0; count = 0;
    while (count < np) {
        int found = 0;
        for (int i = 0; i < np; i++) {
            if (finish[i]) continue;
            int ok = 1;
            for (int j = 0; j < nr; j++) if (need[i][j] > work[j]) { ok = 0; break; }
            if (ok) { for (int j = 0; j < nr; j++) work[j] += alloc[i][j]; finish[i]=1; safe_seq[seq_len++]=i; found=1; count++; }
        }
        if (!found) break;
    }
    if (seq_len == np) {
        printf("Request GRANTED ✓ — System remains safe.\nNew Safe Sequence: ");
        for (int i = 0; i < np; i++) printf("P%d%s", safe_seq[i], i<np-1?" → ":"");
        printf("\n");
    } else {
        printf("Request DENIED ✗ — Would lead to unsafe state. Rolling back.\n");
        for (int j = 0; j < nr; j++) { avail[j]+=req[j]; alloc[pid][j]-=req[j]; need[pid][j]+=req[j]; }
    }
}

typedef struct { int id, size; } Block;

void first_fit(void) {
    int nb, np;
    Block blocks[MAX_BLOCKS];
    int proc_sizes[MAX_PROC];
    int assigned[MAX_PROC], alloc[MAX_BLOCKS];

    printf("\n=== Q15a: First Fit Memory Allocation ===\n");
    printf("Enter number of memory blocks: "); scanf("%d", &nb);
    for (int i = 0; i < nb; i++) {
        printf("  Block[%d] size: ", i); scanf("%d", &blocks[i].size);
        blocks[i].id = i; alloc[i] = -1;
    }
    printf("Enter number of processes: "); scanf("%d", &np);
    for (int i = 0; i < np; i++) {
        printf("  Process P%d size: ", i); scanf("%d", &proc_sizes[i]);
        assigned[i] = -1;
    }

    /* First Fit */
    int free_left[MAX_BLOCKS];
    for (int i = 0; i < nb; i++) free_left[i] = blocks[i].size;

    for (int i = 0; i < np; i++) {
        for (int j = 0; j < nb; j++) {
            if (free_left[j] >= proc_sizes[i]) {
                assigned[i] = j;
                free_left[j] -= proc_sizes[i];
                alloc[j] = i;
                break;
            }
        }
    }

    printf("\nAllocation Result:\n");
    printf("%-10s %-10s %-15s\n", "Process", "Size", "Block Assigned");
    for (int i = 0; i < np; i++) {
        if (assigned[i] == -1)
            printf("P%-8d %-10d NOT ALLOCATED\n", i, proc_sizes[i]);
        else
            printf("P%-8d %-10d B%d\n", i, proc_sizes[i], assigned[i]);
    }
    print_alloc(blocks, nb, alloc, np);
}

/* ================================================================
 * Q18: TLB Simulation
 * ================================================================
 * TLB = Translation Lookaside Buffer (small cache of page→frame mappings)
 * On each access:
 *   TLB Hit  : frame found in TLB (fast path, 1 memory access)
 *   TLB Miss : look up page table, add to TLB (2 memory accesses)
 * Replacement policy: FIFO for TLB
 *
 * Sample Input:
 *   Page table: [2, 8, 3, 1, 9, 5]
 *   Page references: 0 1 2 3 4 0 1 0 3 2
 * ================================================================ */
void tlb_simulation(void) {
    int page_size, n_pages, n_refs;
    int page_table[MAX_PROC];

    printf("\n=== Q18: TLB Simulation ===\n");
    printf("Enter page size (bytes): ");     scanf("%d", &page_size);
    printf("Enter page table size: ");       scanf("%d", &n_pages);
    printf("Enter page table (frame for each page):\n");
    for (int i = 0; i < n_pages; i++) {
        printf("  Page %d → Frame: ", i);  scanf("%d", &page_table[i]);
    }
    printf("Enter number of references: "); scanf("%d", &n_refs);

    /* TLB: FIFO arrays */
    int tlb_page[TLB_SIZE], tlb_frame[TLB_SIZE];
    for (int i = 0; i < TLB_SIZE; i++) { tlb_page[i] = -1; tlb_frame[i] = -1; }
    int tlb_ptr = 0;

    int tlb_hits = 0, tlb_misses = 0;
    double tlb_time = 5.0, mem_time = 100.0;   /* ns */
    double total_time = 0.0;

    printf("\n%-6s %-6s %-8s %-10s %-12s\n",
           "Step", "Page", "TLB", "Frame", "Accesses");
    printf("--------------------------------------------------\n");

    for (int i = 0; i < n_refs; i++) {
        int page; printf("  Ref[%d] page: ", i); scanf("%d", &page);

        /* Search TLB */
        int hit = -1;
        for (int j = 0; j < TLB_SIZE; j++)
            if (tlb_page[j] == page) { hit = j; break; }

        if (hit != -1) {
            tlb_hits++;
            int frame = tlb_frame[hit];
            total_time += tlb_time + mem_time;   /* TLB + 1 memory */
            printf("%-6d %-6d %-8s %-10d %-12s\n", i+1, page, "HIT", frame, "TLB+Mem");
        } else {
            tlb_misses++;
            int frame = page_table[page];
            /* Add to TLB (FIFO) */
            tlb_page[tlb_ptr]  = page;
            tlb_frame[tlb_ptr] = frame;
            tlb_ptr = (tlb_ptr + 1) % TLB_SIZE;
            total_time += tlb_time + 2.0 * mem_time;   /* TLB + 2 memory */
            printf("%-6d %-6d %-8s %-10d %-12s\n", i+1, page, "MISS", frame, "TLB+2×Mem");
        }
    }
    printf("--------------------------------------------------\n");
    double hit_ratio = (double)tlb_hits / n_refs;
    printf("TLB Hits   : %d\nTLB Misses : %d\n", tlb_hits, tlb_misses);
    printf("Hit Ratio  : %.2f%%\n", hit_ratio * 100);
    printf("EAT        = %.2f × %.2f + %.2f × %.2f\n",
           hit_ratio, tlb_time + mem_time,
           1.0 - hit_ratio, tlb_time + 2.0 * mem_time);
    printf("EAT        = %.2f ns\n", total_time / n_refs);
}

Worst Fit Largest block ≥ size Maximises leftover O(n) scan
Best Fit minimises wasted space per allocation, but creates the most unusable tiny fragments overall. First Fit is fastest in practice.
BANKBanker's Algorithm — Safety & Request
▼

Data Structures

  • Available[r] — number of free instances of each resource
  • Max[p][r] — maximum demand of process p for resource r
  • Allocation[p][r] — currently allocated to process p
  • Need[p][r] = Max[p][r] − Allocation[p][r]

Safety Algorithm

  • Work = Available; Finish[i] = false for all i
  • Find i where Finish[i]=false AND Need[i] ≤ Work
  • Work += Allocation[i]; Finish[i] = true
  • Repeat until no such i found
  • Safe iff all Finish[i] = true

Resource Request Algorithm

  • If Request > Need[i] → error (exceeds declared max)
  • If Request > Available → wait (resources not free)
  • Tentatively: Available -= Request; Allocation[i] += Request; Need[i] -= Request
  • Run Safety Algorithm. If safe → grant. Else → roll back.
Time Complexity: O(n² × r) per safety check
n = processes, r = resource types
TLBEffective Access Time & TLB Theory
▼

TLB (Translation Lookaside Buffer)

  • Small, fast associative cache for page-table entries
  • Typical size: 16–512 entries
  • Hit ratio (α) typically 80–99% in practice
  • On TLB hit: 1 memory access (for data)
  • On TLB miss: page table walk + 1 memory access = 2 total
Single-level paging:
  EAT = α(t_TLB + t_mem) + (1-α)(t_TLB + 2·t_mem)
  EAT = t_TLB + (2 - α)·t_mem

k-level paging (no TLB):
  EAT = (k + 1) × t_mem

k-level paging (with TLB):
  EAT = α(t_TLB + t_mem) + (1-α)(t_TLB + (k+1)·t_mem)
With α=0.98, t_TLB=10ns, t_mem=100ns:
EAT = 10 + (2 - 0.98)×100 = 10 + 102 = 112ns (vs 200ns without TLB → 43% speedup)
